## О проекте

> **От базовой структуры к production-практикам: Nest.js, TypeScript и современная backend-разработка**

Этот репозиторий предназначен для комплексного изучения Nest.js с нуля. Основной упор сделан на контроллеры, провайдеры, модули, архитектуру и использование TypeScript для безопасности типов. Каждый проект полностью независим с собственной конфигурацией и структурой приложения.

## Быстрый старт

### Запуск упражнений

```bash
# 1. Клонирование репозитория
git clone <repository-url>
cd -Nest.js_Education-

# 2. Переход в упражнение
cd practical-task/nameProject

# 3. Установка зависимостей и запуск
npm install && npm run dev
```

### Запуск проектов

```bash
cd practical-task/nameProject
npm install && npm run dev
```

## Команды

```bash
# В папке каждого проекта/упражнения
npm run dev        # Запуск dev сервера (http://localhost:3000)
npm run build      # Production сборка
npm run start      # Запуск production-версии
npm run lint       # Проверка ESLint
```
